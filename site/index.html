Git is a free and open source distributed version control system designed to handle everything from small to very large projects speed and efficiency. Git is easy to learn and has a tiny footprint with lightning fast performance. It outclasses SCM tools like Subversion, CVS, Perforce, and ClearCase with features like cheap local branching, convenient staging areas, and multiple workflows.

  • git clone
    When dealing with repository / files in github, make sure

  • git status | check the origin and file's status
  • git checkout [branch] | change branch/owner
  • git add [files]/[.] | add file/all files to be ready-commit state
  • git commit -m "2nd change" | commit the file with comments
  • git push | push the files to github repos online
