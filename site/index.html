Here are some git command that are helpfull to upload and clone repos :

1. git clone [https_link] // get HTTPS Repos URL to local machine
2. git status // cek origin dan file pending
3. git checkout [branch] // ubah branch
4. git status // check origin and file pending
5. git add . // add all files to be commit
6. git commit -m "2nd change" // commit the file
7. git push // push to github database
