SOME USEFUL COMMANDS TO BE WORKIN WITH GITHUB

Git is a free and open source distributed version control system designed to handle everything from small to very large projects speed and efficiency. Therefore, here are some commands that are essentials for Web Developer to be able update and handle their repos easily.

  • git clone [HTTPS_link] | get HTTPS Repos URL to Local Machine
  • git status | check the origin and file's status
  • git checkout [branch] | change branch/owner
  • git add [files]/[.] | add file/all files to be ready-commit state
  • git commit -m "2nd change" | commit the file with comments
  • git push | push the files to github repos online

Browser Sync : browser-sync start --server --directory --files "*"
